Bienvenid@ a la api de Ele-Task

Aquí podrá consultar la información sobre las tareas y la lista de usuarios

Información sobre las tareas

Aquí podrás consultar el número de tareas que están guardadas en nuestra base de datos. Además, podrás ver el número de tareas realizadas, vencidas y realizadas.

Las tareas son privadas, por lo que no podrá visualizar tareas de terceras personas.

Enlace: https://ele-task.srequenafiliu.repl.co/tareas

Lista de usuarios

Aquí podrás consultar los usuarios registrados en nuestra base de datos. Podrá navegar a través de las páginas mediante los enlaces next y previous, además de cambiar el número de usuarios visualizados con el parámetro de consulta size.

El nombre, correo electrónico y contraseña de los usuarios no se podrán ver por motivos de seguridad.

Enlace: https://ele-task.srequenafiliu.repl.co/usuarios
